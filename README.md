# Back-end H-DIC

Django server serving json files

# Installing

```
cd hdic
pipenv install
```

# Running

```
cd hdic
pipenv run ./manage.py runserver
```

# Requesting

You can do GET request at `http://127.0.0.1:8000/search/look`

Example : `http://127.0.0.1:8000/search/look/?search=Ail,CETUXIMAB`
